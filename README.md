# OpenCode Skills Registry

A community-driven catalog of reusable agent instructions, workflows, and capabilities for [OpenCode](https://opencode.ai) and compatible agent runtimes.

## Browse

The registry is browsable at **[opencode.ai/skills](https://opencode.ai/skills)** (or run locally via the skill index app).

## Quick Start

```bash
# Add a skill to your opencode config
opencode skill add <skill-name>

# List available skills
opencode skill list

# Or browse the web index
```
