# commit

Contextual commit messages with structured formatting. Analyzes staged changes and produces conventional commit messages with scope and body.

## Prerequisites

Git

---

## How to Use

When committing changes:
1. Run `git diff --staged` to review changes
2. Analyze commit scope and intent
3. Generate conventional commit message:

```
<type>(<scope>): <short summary>

<body>
```
