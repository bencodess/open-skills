# security-audit

Security review and vulnerability detection. Scans dependencies, code patterns, and configuration for OWASP Top 10 and CVE exposure.

## Prerequisites

Python (for dependency scanning), `npm audit` or `pip audit`

---

## How to Use

When performing a security audit:
1. Scan dependencies for known CVEs
2. Check code for OWASP Top 10 patterns
3. Review configuration for security best practices
4. Report findings with severity levels
