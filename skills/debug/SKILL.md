# debug

Systematic debugging workflow. Reproduces issues, instruments code, analyzes logs, and iterates toward root cause identification with suggested fixes.

## Prerequisites

None

---

## How to Use

When debugging an issue:
1. Reproduce the problem
2. Instrument code with logging
3. Analyze logs and stack traces
4. Narrow down to root cause
5. Suggest and verify fix
