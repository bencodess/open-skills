# accessibility

Accessibility auditing and remediation. Scans for WCAG violations, suggests ARIA attributes, keyboard navigation fixes, and screen reader improvements.

## Prerequisites

Browser, axe-core / Lighthouse

---

## How to Use

When auditing accessibility:
1. Run automated scans (axe, Lighthouse)
2. Check for WCAG violations
3. Fix ARIA attributes and keyboard navigation
4. Test with screen reader
