# performance

Performance optimization analysis. Identifies bottlenecks in code, bundles, rendering, and queries with measurable improvement suggestions.

## Prerequisites

Browser DevTools, Lighthouse, or profiling tools

---

## How to Use

When optimizing performance:
1. Measure current performance (Lighthouse, bundlesize)
2. Identify bottlenecks (render, network, computation)
3. Suggest optimizations with expected impact
4. Verify improvements
