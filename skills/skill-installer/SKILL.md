# skill-installer

Install curated and external skills from GitHub repositories. Automatically discovers and places SKILL.md files in the correct directory.

## Prerequisites

Git, OpenCode

---

## How to Use

When user wants to install a skill from a GitHub repository:

```bash
# clone the skill repo
git clone <repo-url> ~/.opencode/skills/<name>
```
