# memory

AI memory management across sessions. Stores and retrieves context using vector databases, enabling persistent cross-session knowledge for agents.

## Prerequisites

Vector database (or in-memory fallback)

---

## How to Use

When managing AI memory across sessions:
1. Store context with embeddings in vector DB
2. Retrieve relevant context on session start
3. Update context with new information
4. Manage token limits with summarization
