# deep-research

Structured deep research skill with human-in-the-loop control. Multi-step investigation with source tracking, summarization, and citation management.

## Prerequisites

Web search access, Python (optional for source tracking)

---

## How to Use

When performing deep research:
1. Formulate research questions
2. Search multiple sources iteratively
3. Track citations with source metadata
4. Summarize findings with human-in-the-loop checkpoints
