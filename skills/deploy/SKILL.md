# deploy

Deployment automation across environments. Manages build, staging, health checks, rollbacks, and environment-specific configuration injection.

## Prerequisites

Access to deployment target (server, cloud, etc.)

---

## How to Use

When deploying an application:
1. Build the artifact
2. Run health checks on staging
3. Configure environment variables
4. Deploy with rollback capability
5. Monitor post-deploy
