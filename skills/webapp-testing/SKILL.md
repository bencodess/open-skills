# webapp-testing

Test local web apps with Playwright. Uses Python Playwright scripts with server lifecycle helper. Inspects rendered pages before acting with discovered selectors.

## Prerequisites

Python 3.8+, Playwright (`pip install playwright`), `playwright install chromium`

---

## How to Use

When testing a local web app:
1. Start the dev server
2. Write or run Playwright test scripts
3. Use server lifecycle helper for setup/teardown
