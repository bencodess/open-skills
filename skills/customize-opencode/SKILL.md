# customize-opencode

Edit or create opencode's own configuration files including opencode.json, .opencode/ files, agents, subagents, skills, plugins, MCP servers, and permission rules.

## Prerequisites

OpenCode installed

---

## How to Use

When user asks to modify opencode configuration, edit the relevant config file in `~/.config/opencode/` or the project's `.opencode/` directory.
