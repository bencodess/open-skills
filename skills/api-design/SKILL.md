# api-design

REST and GraphQL API design patterns. Suggests endpoint structures, request/response schemas, error handling, versioning, and OpenAPI specification generation.

## Prerequisites

None

---

## How to Use

When designing an API:
1. Identify resources and operations
2. Design RESTful or GraphQL schema
3. Define request/response shapes
4. Plan error handling and versioning
5. Generate OpenAPI spec
