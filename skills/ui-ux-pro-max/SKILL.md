# ui-ux-pro-max

Comprehensive UI/UX design intelligence with searchable database of 67 styles, 96 color palettes, 57 font pairings, and 99 UX guidelines.

## Prerequisites

Python 3.8+

---

## How to Use

When user requests UI/UX work (design, build, review), run the design system search:

```bash
python3 skills/ui-ux-pro-max/scripts/search.py "<query>" --design-system
```
