# ci-cd

CI/CD pipeline setup and maintenance. Configures GitHub Actions, GitLab CI, or other runners with caching, parallel jobs, and deployment stages.

## Prerequisites

GitHub account (for GitHub Actions) or other CI provider

---

## How to Use

When setting up CI/CD:
1. Detect project language and test framework
2. Create pipeline config with lint, test, build stages
3. Add caching for dependencies
4. Configure deployment stage if applicable
