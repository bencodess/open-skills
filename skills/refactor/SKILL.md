# refactor

Code refactoring patterns and migrations. Identifies code smells, plans incremental refactors, and executes safe transformations with test validation.

## Prerequisites

Tests (to validate safety)

---

## How to Use

When refactoring code:
1. Identify code smells (long functions, duplication, tight coupling)
2. Plan incremental refactors
3. Execute changes with test validation at each step
