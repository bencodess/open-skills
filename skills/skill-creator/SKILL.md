# skill-creator

Guide agents through building well-structured SKILL.md files with proper YAML frontmatter, descriptions, and usage instructions.

## Prerequisites

None

---

## How to Use

When user wants to create a new skill, guide them through:
1. Choose a kebab-case name
2. Write YAML frontmatter (name, description)
3. Add usage instructions with Prerequisites, How to Use, and Examples sections
